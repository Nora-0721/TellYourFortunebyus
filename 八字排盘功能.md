AI 面相 + 八字合参算命项目 —— 正式开发说明书（可直接交付 GPT）
一、项目现状与老师要求
1. 现有功能
用户上传照片，选择性别，选择年龄区间。
后端 CNN 解析面部特征，输出 JSON 坐标 / 特征。
调用通义千问 API 一次生成三段文本：
第一段：面相分析（展示在第一页）
第二段：正缘性格、职业、理想型分析（展示在第二页）
第三段：正缘面貌描述 Prompt（不展示，给文生图模型用）
前端左右翻页按键不变，仅切换内容。
2. 老师新增要求
引入开源项目：https://github.com/zengzhan/rili-bazi
将该项目的八字排盘、五行、十神、神煞、命理规则作为 Skill（专业技能） 集成进系统。
让算命结果更专业、更准确，符合传统命理逻辑。
允许新增输入项、新增展示页面，不破坏原有功能。

当前结构：

第 2 页：正缘分析

三、Skill 集成说明（老师要求的核心部分）
1. 什么是 Skill？怎么加？
Skill不仅包含从rili-bazi抄来的工具函数（你已完成的bazi_skill.py，负责八字算法计算），还包含固定的专家解读规范（即我提供的命理专家角色、输出结构、Prompt模板）。

新建标准Skill目录：/myskills/bazi_expert/，包含3个文件（与项目现有Skill规范保持一致）：
    - skill.md：说明该Skill用途（对八字命盘进行专业专家级解读，依赖bazi_skill输出的命盘JSON）；
    - prompt.txt：将我提供的完整命理专家Prompt（角色设定、特别指令、输出结构、Task_Instructions等）完整复制进去，不做任何修改；
    - __init__.py：编写简单调用函数（get_bazi_expert_prompt），用于读取prompt.txt中的内容，供LLM调用。

修改generate_bazi_analysis函数（仅修改Prompt调用部分，不改动其他逻辑）：
    - 不再使用原有简单Prompt，而是调用新建的bazi_expert_skill中的get_bazi_expert_prompt()，获取专业Prompt；
    - 将bazi_skill计算出的完整命盘数据（四柱、五行、旺衰、藏干等），自动填充到Prompt中的<User_Bazi_Data>位置，实现全自动专家解读。

 
2. 必须从老师项目中抄的功能
从 rili-bazi 复制以下核心逻辑：
八字排盘逻辑
输入：出生年月日时 + 性别
输出：年柱、月柱、日柱、时柱、八字字符串
五行计算
输出：金木水火土旺衰
十神解析
输出：日主、比肩、财官印等
神煞基础信息（可选，用于丰富文本）
年龄自动计算
根据生日 → 计算周岁年龄
