## 2. 功能改动说明（按模块）

### 2.1 `mainsite/bazi_skill.py`（新增）
- 新增八字计算工具：根据出生日期/时辰生成四柱、五行、十神、年龄区间。
- 增加流年与大运关系输出字段（含今年/明年干支、合冲刑害关系）。
- 输出结构化 profile，供视图层和 LLM 解读复用。

### 2.2 `myskills/bazi_expert/*`（新增）
- 新增八字专家 Skill 目录与 prompt 资源。
- `get_bazi_expert_prompt()` 用于从本地 prompt 文件读取模板。
- 用于将命盘 JSON 注入专家提示词，约束 LLM 产出结构化文本。

### 2.3 `ai/__init__.py`
- 引入 bazi expert skill prompt。
- 新增/强化 `generate_bazi_analysis`：
	- 接收命盘 JSON。
	- 使用 Skill Prompt 调 DashScope 生成八字文本。
	- 本地兜底文案、去模板回声、去重复流年条目、关键行补齐。
- 调整正缘生图提示词年龄逻辑：由“年龄段文本”改为按年龄阶段策略。

### 2.4 `mainsite/views.py`
- 引入 `build_bazi_profile` 与 `generate_bazi_analysis`。
- 新增会话缓存与重算策略：
	- 上传新图或更新生辰后清空旧八字/正缘缓存。
	- 八字 profile 缺字段时自动重算。
	- 本地兜底结果在有 API key 时可触发刷新。
- 增加八字文本分段与 HTML 格式化：
	- 流年干支
	- 旺衰与定局
	- 战略箴言

### 2.5 `templates/input_ch.html` 与 `templates/input_phone.html`
- 输入项由“手动年龄段”改为“生辰（日期 + 时辰）”。
- 保留原上传照片、性别等主流程，增加表单布局适配。

### 2.6 `templates/showpage.html`
- 展示页新增 3 个八字分段面板（流年/旺衰/箴言）。
- 翻页逻辑由两页扩展为五页顺序切换：
	- 本人报告 -> 八字流年 -> 旺衰定局 -> 战略箴言 -> 正缘报告
- 补充八字文本样式类，避免与现有正文样式冲突。

### 2.7 `八字排盘功能.md`（新增）
- 记录八字功能开发目标、数据流、页面结构与约束，作为需求说明文档。

## 3. 同学合并时重点关注（减少冲突成本）

### 3.1 高冲突文件
- `mainsite/views.py`
- `templates/showpage.html`
- `templates/input_ch.html`
- `templates/input_phone.html`

### 3.2 建议合并顺序
1. 先合并后端：`mainsite/bazi_skill.py`、`myskills/bazi_expert/*`、`ai/__init__.py`、`mainsite/views.py`。
2. 再合并前端模板：`templates/*`。
3. 最后处理可忽略文件（`__pycache__`、`db.sqlite3`）。

### 3.3 冲突处理建议
- 若文生图分支也改了 `showpage.html`/`views.py`，优先保留：
	- 八字分段面板与分页状态机逻辑。
	- 同学文生图按钮和接口调用逻辑。
- `__pycache__` 直接丢弃冲突版本并重新运行生成即可。
- `db.sqlite3` 以各自本地开发库为准，不作为功能变更依据。
