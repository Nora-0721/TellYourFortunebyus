一、项目现状与老师要求
1. 现有功能
用户上传照片，选择性别，选择年龄区间。
后端 CNN 解析面部特征，输出 JSON 坐标 / 特征。
调用通义千问 API 一次生成三段文本：
第一段：面相分析（展示在第一页）
第二段：正缘性格、职业、理想型分析（展示在第二页）
第三段：正缘面貌描述 Prompt（不展示，给文生图模型用）
前端左右翻页按键不变，仅切换内容。
2. 老师新增要求
引入开源项目：https://github.com/zengzhan/rili-bazi
将该项目的八字排盘、五行、十神、神煞、命理规则作为 Skill（专业技能） 集成进系统。
让算命结果更专业、更准确，符合传统命理逻辑。
允许新增输入项、新增展示页面，不破坏原有功能。

三、Skill 集成说明（老师要求的核心部分）
1. 什么是 Skill？怎么加？
Skill不仅包含从rili-bazi抄来的工具函数（你已完成的bazi_skill.py，负责八字算法计算），还包含固定的专家解读规范（即我提供的命理专家角色、输出结构、Prompt模板）。

新建标准Skill目录：/myskills/bazi_expert/，包含3个文件（与项目现有Skill规范保持一致）：
    - skill.md：说明该Skill用途（对八字命盘进行专业专家级解读，依赖bazi_skill输出的命盘JSON）；
    - prompt.txt：将我提供的完整命理专家Prompt（角色设定、特别指令、输出结构、Task_Instructions等）完整复制进去，不做任何修改；
    - __init__.py：编写简单调用函数（get_bazi_expert_prompt），用于读取prompt.txt中的内容，供LLM调用。

修改generate_bazi_analysis函数（仅修改Prompt调用部分，不改动其他逻辑）：
    - 不再使用原有简单Prompt，而是调用新建的bazi_expert_skill中的get_bazi_expert_prompt()，获取专业Prompt；
    - 将bazi_skill计算出的完整命盘数据（四柱、五行、旺衰、藏干等），自动填充到Prompt中的<User_Bazi_Data>位置，实现全自动专家解读。

 
2. 必须从老师项目中抄的功能
从 rili-bazi 复制以下核心逻辑：
八字排盘逻辑
输入：出生年月日时 + 性别
输出：年柱、月柱、日柱、时柱、八字字符串

年龄自动计算
根据生日 → 计算周岁年龄

4. 这个 JSON 如何喂给通义千问 LLM
原有 LLM 调用不动，新增一次独立 LLM 调用，专门生成八字文本：
原有流程（完全不变）照片 → CNN → LLM → 面相 + 正缘 + 文生图 Prompt
新增流程（Skill 部分）生日 → Skill（排盘函数） → 上面的 JSON → LLM → 八字分析文本
5. 多出来的输出如何与原有输出协调、不乱套
规则：
原有三段输出：完全保留，结构不变、字段不变、顺序不变
八字文本：作为第四个独立输出项
前端只负责 “取对应页的内容”，互不干扰：
第 1 页 → 取原有第 1 段（面相）
第 2 页 → 取新增第 4 段（八字）
第 3 页 → 取原有第 2 段（正缘）
原有第 3 段（文生图 Prompt）继续隐藏，不展示
这样结构永远不乱。
四、年龄输入改动与对 Prompt 的影响
1. 前端输入改动
删除原来的 “手动选择年龄区间”
改为：用户输入生辰八字（年月日时）
后端通过 Skill 自动计算：
年龄 age
5 岁区间 age_range（如 26–30 岁）

2. 对文生图 Prompt 的直接影响
原有 Prompt 示例：

新 Prompt 自动替换为：
plaintext

例如用户 28 岁：
plaintext
理想对象年龄在26-30岁之间，属于青年，外貌气质符合该年龄段，与用户年龄适配。
3. 为什么不用 “青年 / 中年 / 老年”？
因为文生图模型对精确区间理解远强于模糊词汇，生成人物更稳定、更符合预期，因此采用：年龄 = 自动计算 → 5 岁区间 → 填入 Prompt

东方十二星次（中式十二星座）对应文案规则
 
1. 星纪-摩羯宫/农历十二月初七至次年正月初五/“上当星纪，下裂坤维。”——唐·王勃《广州宝庄严寺舍利塔》/日月五星轮回的终始，故称“星纪”。作为十二星次的出发点，有根基的含义。信念坚强而勤奋，会朝着自己的目标努力不懈。生性温柔，但若发起脾气，也是势不可挡的。
2. 玄枵-水瓶宫/农历正月初六至二月初三/“自须女八度至危十五度为玄枵，於辰在子，齐之分野，属青州。”——《晋书·天文志上》/有种子的含义，兼具发展性和神秘感。好奇心旺盛，性格开朗、乐观，兴趣广泛。但同时缺乏耐心，善变，对一件事物的热情不会太持久。
3. 娵訾-双鱼宫/农历二月初四至三月初五/“及其亡也，岁在娵訾之口。”——《左传·襄公三十年》/代表植物的核，意指中心。个性强，注重自我。外表冷漠内心却十分热情，而且具有恒心和毅力，一旦决定了某件事，便会以坚强的意志坚持到底。不过这种坚强可能会导致过于自我。
4. 降娄-白羊宫/农历三月初六至四月初四/“自奎五度至胃六度为降娄，於辰在戌，鲁之分野，属徐州。”——《晋书·天文志上》/代表植物的茎，像茎向植物无私地输送养分一般，是一个为他人无私奉献的人。性格直率，乐于助人，但容易过于听从别人的想法而失去主见。
5. 大梁-金牛宫/农历四月初五至五月初五/“岁在大梁，将集天行。”——《国语·晋语四》/象征植物积蓄能量而生长，有结实的意思。头脑灵活，有先见之明，善于规划。但也会比较现实而精于算计。
6. 实沈-双子宫/农历五月初六至六月初五/“岁在大梁，将集天行，元年始受实沈之星也。”——《国语·晋语四》/代表伸缩自如的柔软性，象征植物的枝条。有韧性，善于随机应变。但有时会缺乏踏实的努力，天赋需要真正的磨练才能转化为才能。
7. 鹑首-巨蟹宫/农历六月初六至七月初六/“天文家‘朱鸟’，乃取象於鹑。故南方朱鸟七宿，曰鹑首、鹑火、鹑尾是也。”——宋·沈括《梦溪笔谈·象数一》/象征着宁静。沉稳温和，总给人很有内涵而且略有神秘感的印象。外冷内热，有包容力和涵养。但有时会过于计较。
8. 鹑火-狮子宫/农历七月初七至八月初七/“岁在鹑火，是以卒灭。”——《左传·昭公八年》/代表不死鸟的心脏，具有燃烧的生命力。精力充沛，热情四溢，毫不做作。但也是典型的忽冷忽热的性格。
9. 鹑尾-室女宫/农历八月初八至九月初七/“岁在寿星及鹑尾，其有此土乎！”——《国语·晋语四》/代表着坚实地生长在大地中的根。性格会如同土地中的根一样，坚强而深不可测。外表冷峻，内心温柔，想象力丰富，但有时会不切实际。
10. 寿星-天秤宫/农历九月初八至十月初七/“东宫则析木之津，寿星之野。”——唐·杨炯《浑天赋》/福寿双全之命格，表面上慵懒，但一旦有了目标，就会充满热情地开始实干。但这种热情，往往难以持久，而且不喜欢团体活动。
11. 大火-天蝎宫/农历十月初八至十一月初八/“大火谓之大辰。”——《尔雅·释天》/代表早晨温暖的阳光。给别人的感觉十分温暖，性格安宁恬静而深得大家喜爱，爱家顾家。但容易优柔寡断。
12. 析木-射手宫/农历十一月初九至十二月初六/“日月会於析木兮，重阴凄而增肃。”——晋·傅玄《大寒赋》/代表着拦截天河的木栅。意志坚强，直面逆境，勤奋上进。但有时一腔热血和不喜欢束缚也意味着固执。
